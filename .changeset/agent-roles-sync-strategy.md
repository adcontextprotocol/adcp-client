Adopt the shared `.agents/roles/` agent-roles layout from
`adcontextprotocol/adcp#4500`. `.agents/roles/` is now the single source
of truth for all 28 subagent prompts (10 short triage checkers, 9 long
`-deep` design advisors, 9 long-only specialty advisors).
`.claude/agents/` and `.codex/agents/` are generated by
`scripts/import-claude-agents.mjs` — run `npm run sync:agents` after
editing any source role. A new CI workflow
(`.github/workflows/validate-agent-roles-sync.yml`) fails any PR where
`.claude/agents/` has drifted from `.agents/roles/`. `.codex/` is
gitignored so each contributor regenerates locally.
